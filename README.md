# Docker-Composer

## Overview
Exercises with Docker Compose: multi-service setup, service dependencies, networks, and volumes. Hyperskill project.

## Tech
- Docker
- Docker Compose

## How to run

```bash
docker compose up -d
# visit any exposed ports defined in the compose file
docker compose down -v
```

## What this demonstrates
- Service declaration and health dependencies
- Named volumes & networks
- Environment variables and .env
